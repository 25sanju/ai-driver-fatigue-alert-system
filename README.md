# 🚗 Driver Drowsiness Detection System

## 📌 About the Project
This project detects driver drowsiness in real-time using computer vision.  
It monitors eye closure and yawning to detect fatigue and gives voice alerts.

## ⚙️ Technologies Used
- Python
- OpenCV
- MediaPipe
- Windows Speech API

## 🚀 Features
- Real-time face detection
- Eye closure detection
- Yawning detection
- Voice alert system
- Fatigue scoring system

## ▶️ How to Run
pip install opencv-python mediapipe
python main.py

## 🎯 Output
- NORMAL / DROWSY / CRITICAL detection
- Voice warning alerts
